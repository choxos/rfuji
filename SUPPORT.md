# Support

Use the following channels for help with `rfair`:

- Bug reports: open a GitHub issue using the bug report template.
- Feature requests or metric suggestions: open a GitHub issue using the
  feature request template.
- General questions: use GitHub Discussions when available, or open a
  question issue if the question affects package behavior or
  documentation.
- Security issues: follow `SECURITY.md` and do not open a public issue.

For reproducible help, include the identifier being assessed, the metric
version, the code you ran, and
[`sessionInfo()`](https://rdrr.io/r/utils/sessionInfo.html).
