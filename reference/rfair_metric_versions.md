# List the metric versions bundled with rfair.

List the metric versions bundled with rfair.

## Usage

``` r
rfair_metric_versions()
```

## Value

Character vector of available metric versions (e.g. "0.8").

## Examples

``` r
rfair_metric_versions()
#>  [1] "0.8"                 "0.5"                 "0.5ssv2"            
#>  [4] "0.5ss"               "0.5env"              "0.7_software"       
#>  [7] "0.7_software_cessda" "0.6a2a"              "0.4"                
#> [10] "0.3"                 "0.2"                
```
